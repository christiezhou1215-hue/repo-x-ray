# Contributing

Thanks for helping Repo X-Ray explain code more clearly.

Useful contributions include:

- Real examples where an explanation was confusing, too technical, or incorrect
- Improvements that make explanations shorter and clearer
- Better handling of different languages, frameworks, and repository layouts
- Installation fixes for supported Codex environments

Keep changes focused. Explain the user problem, the behavioral change, and how you tested it. Avoid adding generic programming knowledge that Codex already knows.
